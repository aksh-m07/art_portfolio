# Aksh Malik Art Portfolio

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## About

Welcome to the portfolio of Aksh Malik, showcasing photography and digital art. The photography collection focuses on wildlife, capturing moments with birds, butterflies, spiders, and larger animals like tigers and deer. The art collection includes character illustrations, superheroes, and original pieces. The site offers an interactive gallery with carousel and grid views, touch/swipe navigation, and organized collections for easy browsing.
